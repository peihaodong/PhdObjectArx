Description: 
This application implements a command called ADDPALETTE. It will create a sample PaletteSet.

To run the sample:
Please add the References ZwManaged.dll and ZwDatabaseMgd.dll from your ZWCAD installation directory and then Build the application.

To use AddEnts.DLL:
1. Start ZWCAD and open a new drawing.
2. Run NETLOAD command and load the AddPaletteSet.dll file from the \bin subfolder.
3. Execute the ADDPALETTE command.
