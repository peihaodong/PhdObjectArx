Description: 
This application implements a command called ADDENTS. It will create a circle and a red line in the drawing.

To run the sample:
Please add the References ZwManaged.dll and ZwDatabaseMgd.dll from your ZWCAD installation directory and then Build the application.

To use AddEnts.DLL:
1. Start ZWCAD and open a new drawing.
2. Run NETLOAD command and load the AddEnts.DLL file from the \bin subfolder.
3. Execute the ADDENTS command.
