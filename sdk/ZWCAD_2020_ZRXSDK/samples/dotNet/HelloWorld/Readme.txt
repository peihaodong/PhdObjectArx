Description: 
This application implements a command called HelloWorld. It will write a massege to the command window.

To run the sample:
Please add the References ZwManaged.dll and ZwDatabaseMgd.dll from your ZWCAD installation directory and then Build the application.

To use HelloWorld.dll:
1. Start ZWCAD and open a new drawing.
2. Run NETLOAD command and load the HelloWorld.dll file from the \bin subfolder.
3. Execute the HelloWorld command.
